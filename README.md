Stockmate is a desktop application designed to assist photographers in streamlining the process of uploading photos to multiple stock services and social media platforms. It focuses on efficient management of keywords, tags, and titles to optimize the submission workflow.

## Tech Stack üõ†Ô∏è

- **Core Framework:** [Tauri](https://tauri.app/) (Rust backend + Webview frontend)
- **Frontend:** [Svelte](https://svelte.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Package Manager:** [pnpm](https://pnpm.io/)

### Development Environment Note

**Important:** This project is primarily developed on **Windows**. While it should work on macOS and Linux, I cannot guarantee a seamless experience or provide support for those platforms. If you encounter any issues on macOS or Linux, you will need to resolve them yourself.

## Contributing

If you're interested in making Stockmate better, you can follow these steps:

1.  **Fork** the repository.
2.  **Clone** your forked repository: `git clone <your-fork-url>`
3.  Install dependencies: `pnpm i`
4.  Start the development server: `pnpm tauri dev`
5.  Make your **edits**.
6.  Submit a **Pull Request**.

If you want to report a bug or suggest a feature, please open an issue on the GitHub repository.
